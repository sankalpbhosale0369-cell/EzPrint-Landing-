# EzPrint Windows Software Download Landing Page

This is the official landing page for the EzPrint Windows desktop application. 

## Overview

EzPrint provides a streamlined, hassle-free printing experience. This repository contains the source code for the landing page where users can download the desktop software installer.

## Project Structure

- `src/`: React source code (Landing Page, Layouts)
- `public/downloads/`: Contains the `EzPrint-Setup.exe` installer
- `index.html`: Main entry point
- `vite.config.ts`: Build configuration

## Getting Started

### Prerequisites

- Node.js & npm installiert

### Local Development

1. **Install dependencies:**
   ```sh
   npm install
   ```

2. **Run dev server:**
   ```sh
   npm run dev
   ```

3. **Build for production:**
   ```sh
   npm run build
   ```
